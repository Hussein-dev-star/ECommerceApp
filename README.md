# 🛒 ECommerceApp - ASP.NET Core MVC Application

مشروع متجر إلكتروني متكامل مبني باستخدام نمط المعمارية **MVC** في فريمورك **ASP.NET Core**. تم تصميم المشروع ليكون دليلاً عملياً شاملاً لتطبيق عمليات الـ CRUD وإدارة الحسابات والصلاحيات بأعلى المعايير الاحترافية.

## 🚀 الميزات الأساسية للمشروع (Features):
- **معمارية MVC النظيفة:** فصل كامل للمسؤوليات بين الـ Models والـ Views والـ Controllers لتسهيل صيانة وتطوير الكود.
- **إدارة قاعدة البيانات (Data Layer):** ربط كامل بـ **SQL Server** حقيقي باستخدام **Entity Framework Core** والاعتماد على الـ Migrations لتوليد الجداول.
- **نظام لوحة التحكم للأقسام والمنتجات (Dynamic CRUD):** لوحة تحكم كاملة تتيح للأدمن إضافة، تعديل، حذف، وعرض الأقسام والمنتجات ديناميكياً من الشاشة.
- **نظام الحسابات والصلاحيات الحقيقي (Identity Auth):** تأمين المتجر بالكامل باستخدام **ASP.NET Core Identity** لتوفير تسجيل دخول وإنشاء حسابات حقيقية مشفرة في قاعدة البيانات مع دعم تعدد الأدوار (Admin و Customer) لحماية أزرار التحكم وعمليات الحذف.
- **سلة المشتريات التفاعلية (Shopping Cart):** سلة مشتريات ذكية مبنية على الـ **Session** لحفظ المنتجات مؤقتاً وحساب إجمالي الفاتورة أوتوماتيكياً للزبون.
- **البحث والفلترة الذكية (Search & Category Filter):** شريط بحث سريع يبحث في الأسماء والأوصاف، مع أزرار فلترة ديناميكية تعرض المنتجات حسب القسم المختار.
- **رفع الصور الحقيقية (Image Upload):** إمكانية رفع صور حقيقية للمنتجات من الجهاز وحفظها ومعالجتها برمجياً بأمان داخل مجلد الـ `wwwroot`.
- **واجهة مستخدم عصرية (UI Layer):** تصميم جذاب وفخم متوافق مع كافة الشاشات باستخدام الـ **Bootstrap** وتطوير صفحات العرض عبر الـ **Razor Views**.

## 🛠️ التكنولوجيات المستخدمة (Tech Stack):
- C# / .NET
- ASP.NET Core MVC
- Entity Framework Core
- Microsoft SQL Server
- ASP.NET Core Identity
- Bootstrap 5 / HTML5 / CSS3

## 💻 طريقة التشغيل محلياً (How to Run):
1. قم بعمل `Clone` للمشروع على جهازك.
2. تأكد من تعديل الـ `Connection String` في ملف `appsettings.json` ليتوافق مع اسم سيرفر الـ SQL Server الخاص بك.
3. افتح الـ Package Manager Console وقم بتشغيل الأوامر التالية لتوليد قاعدة البيانات تلقائياً:
   ```bash
   Add-Migration InitialCreate
   Update-Database
